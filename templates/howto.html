    (tanıtım görsellerine tıklayarak daha yakından inceleyebilirsiniz)
  • Adım - 1 : (Tek Seferlik)
    Python ve VSCode indiriniz.
  • Adım - 2 : (Tek Seferlik)
    windows tuşuna basıp arama butonuna cmd yazınız. python kurulumu sonrasında pip kullanarak bir takım modüller indirmelisiniz.. komut istemine, sırasıyla aşağıdaki yazıları yapıştırınız, her bir pip kurulumu tamamlandığında diğerini yapıştırarak kurunuz.
    pip install binance
    pip install numpy
    pip install y
  • Adım - 3 : (Tek Seferlik)
    Binance üzerinden apikey ve secret key oluşturunuz, bu keyler, api ile binance'a bağlanmanızı ve cüzdanınızı yönetmenizi sağlayacaktır. İşlem yapabilmek adına gerekli izinleri vermeyi ve ip'nizi tanımlamayı unutmayınız. İP'nizi internet üzerinden sorgulayarak öğrenebilirsiniz.
  • Adım - 4 : (Tek Seferlik)
    Çift taraflı işlem yapabilmeyi etkinleştirmelisiniz (BUY/SELL BOTH). Size ilk atılan kodu, apikey ve secretkeyi görselde gösterildiği gibi ekledikten sonra çift tıklayarak çalıştırınız. Script çalışacak, telefon arayüzünüz değişecek, api üzerinden hesabınızın alım satım yapabilme iznine erişeceksiniz. Kod, görseldekiyle aynıdır. Defaulttur.
  • Adım - 5 : (Haftada Bir Güncellenmeli)
    Artık aldığınız kodu VSCode üzerinden açmalı ve botunuza kimliğinizi tanımlamalısınız. APIKEY ve SECRETKEY'inizi görselde gösterildiği gibi yapıştırınız.
  • Hesabınız API Kullanımına Hazır, Artık Botu Çalıştırmak İçin :
    VSCode'u açınız, terminal'e görselde gösterildiği gibi ulaşınız ve aşağıdaki yazıyı yazınız.
    python api_name.py
