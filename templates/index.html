  • Api Nedir?
    Api, telefon ya da bilgisayarda kullandığınız arayüzden farklı olarak, sunucuya kodlar aracılığıyla bağlanıp işlemler yapmamızı sağlayan ve kodlardan oluşan arayüzdür.
    Not: Binance API, binance'ın kullanmamız için bize sunduğu bir yöntemdir. API, ban yemenize neden olmaz.
    Binance API Nasıl Çalışır?
    Sunucuya bağlanabilmek için binance üzerinden apikey ve apisecretkey üretirsiniz. gerekli izinleri yine binance üzerinden sağlarsınız. ürettikten sonra varolan scriptinizle sunucuya, bilgisayarınız üzerinden istekler atarsınız. attığınız isteklerle de işlemleriniz gerçekleşir.
  • Neden Binance Api Kullanmalısınız?
    İşlemleri elden yapmanız yerine, sistem üzerinden anlık taramalarla, otomatik olarak yapmanızı sağlar. Stratejinize sadık kalmanızı, duygusal karar vermemenize yardımcı olur. Uyurken işlem yapabilmenizi sağlar. Anlık değer hesaplayabilir, o an hesaplayamayacağınız indikatörleri anlık verileri çekerek hesaplar. Ayrıca, izinizi belli ettirmezsiniz. Piyasa yapıcılar ağırlıklı olarak psikolojinizden ve board'ı okumanızdan fikir alarak işlem yapar.
    Api For You Ne Sunar?
    Aklınızdaki stratejiyi manuel olarak yapmak yerine api olarak size sunar. yani stratejinizi kod olarak yazar. böylece al-sat'larınızı bottan otomatik yapabilirsiniz. mail üzerinden de bilgilendirilirsiniz. ancak şunu bilmelisiniz ki biz size sadece kodları ve açıklamalarını atacağız. kodunuzu bağlamak ve kullanmak bizim sorumluluğumuzda değildir.
    Not: kodlar sürekli güncelleme alırlar bu nedenle problemlerle karşılaşmanız olasıdır. sorumluluk kabul etmemekle beraber. muhtemel kod kullanım süresinin 1-1.5 ay olduğunu peşinen söyleyelim.
